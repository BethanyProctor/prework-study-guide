HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The base element defines the URL for a webpage.
  • The style element defines the CSS styles associated with a website.
  • The link element connects an external resoursce to the HTML doc.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href specifies the location (URL) of the external resource, using a relative path.
  • The meta element defines metadata such as the character set, description, keywords, authorm and viewport.
  • The script element embeds client-side scripts.

CSS

  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A selector defines the element or attributes to which rules, or declarations, will apply.
  • declarations contain two components: the CSS property we want to apply (such as the color) and the value of the property (such as blue).
  • Examples of CSS rules: height, width, margin-left or -right, font-size, text-align, background-color, text size.
  • display: block assigns each item to start on a new line.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • How to save our work to GitHub:
    1. git add -A
    2. git commit -m "your message"
    3. git pull origin main
    4. git push origin branch-name

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Use keyword "const" to create variables that can't be reassigned a value.
  • After the "const" keyword, give the variable its unique name, followed by = as an assignment operator.
  • Use the strict equality operator === to check if two values are equal and to get a Boolean result true if they are equale and false if they are not equal.
  • An array is a single variable that is used to hold a group of data that is usually related in some way.
  • Array index: the location of the data item in the array (starting at 0 and increasing by 1). For example, const name [0, 1, 2]
  • A "for" loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • Use "let" followed by a unique variable name and = to assign the specific data that we want stored in our variable, and be able to reassign the value over time.
  • A "for" loop has three important statements: starting point for the loop (such as let x = 0), the condition, and the final statement that allows the array to iterate over each item (such as x++).
  • A function is a set of instructions that tells the computer how to perform a certain task, but does not automatically execute when the JS file is run.
  • A function declaration uses the "function" keyword followed by the function name and () and {} to define it.
  • In the {} of a function declaration, include the code block with console.log(); with the instructions.
  • Call the function by writing the function name followed by ();
